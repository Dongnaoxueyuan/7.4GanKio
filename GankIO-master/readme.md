基于 Material Design，使用 MVP + RxJava + Retrofit + ButterKnife 开发干货集中营客户端，接口由[干货集中营](http://gank.io/)提供，非常感谢~

![运行效果](gank.gif)

目前 MVP 模式代码已全部合并到 `master` 分支，若要查看非 MVP 模式的代码请在 git 的提交记录中找到 tag `v1.0.0`

如果觉得本项目对您有帮助，欢迎给个 Star 或者分享给小伙伴哦~
